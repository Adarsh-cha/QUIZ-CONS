# Java Quiz Console Application

This is a simple Java console-based quiz program.  
It contains 5 questions with 4 options each.  
If the user enters a wrong answer, the program asks again.  
At the end, the score is displayed.

## Features
- 5 questions
- 4 options per question
- Repeats question if answer is wrong
- Shows final score

## How to Run
1. Open the project in any Java IDE or terminal
2. Compile the file:
   javac QuizApp.java
3. Run the program:
   java QuizApp
